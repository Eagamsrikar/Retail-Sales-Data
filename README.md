# Exploratory Data Analysis (EDA) on Retail Sales Data

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a retail sales dataset to identify patterns, trends, and business insights that help retail organizations make data-driven decisions.

The analysis includes data cleaning, statistical analysis, customer behavior study, product performance evaluation, time-series analysis, and visualization of sales trends.

---

## 🎯 Objectives

* Load and preprocess retail sales data.
* Perform descriptive statistical analysis.
* Analyze sales trends over time.
* Study customer demographics and purchasing behavior.
* Evaluate product category performance.
* Visualize insights using charts and graphs.
* Provide actionable business recommendations.

---

## 📂 Dataset Description

The dataset contains retail transaction details such as:

* **Date** – Transaction date
* **Customer ID** – Unique customer identifier
* **Gender** – Customer gender
* **Age** – Customer age
* **Product Category** – Category of purchased product
* **Quantity** – Number of items purchased
* **Price per Unit** – Cost per item
* **Total Amount** – Total transaction value

## ⚙️ Project Workflow

### 1️⃣ Data Loading and Cleaning

* Imported dataset using Pandas.
* Converted date columns to datetime format.
* Checked missing values and duplicates.
* Cleaned dataset for analysis.

### 2️⃣ Descriptive Statistics

* Calculated mean, median, mode, and standard deviation.
* Generated statistical summaries.

### 3️⃣ Time Series Analysis

* Analyzed monthly sales trends.
* Identified peak sales periods.

### 4️⃣ Customer Analysis

* Gender distribution analysis.
* Age distribution visualization.
* Spending behavior comparison.

### 5️⃣ Product Analysis

* Sales performance by product category.
* Quantity sold analysis.

### 6️⃣ Visualization

Created multiple visualizations:

* Line charts
* Bar charts
* Histograms
* Box plots
* Correlation heatmaps

### 7️⃣ Business Recommendations

Generated insights to improve:

* Marketing strategies
* Inventory planning
* Customer engagement
* Pricing decisions

---

## 📊 Key Insights

* Certain product categories contribute the majority of revenue.
* Sales show time-based trends indicating seasonal demand.
* High-value customers significantly impact total revenue.
* Customer demographics influence purchasing behavior.
